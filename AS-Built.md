# AS-BUILT Documentation

## Project Overview
- **Name:** ccsp-quiz-flashcards
- **Version:** 0.1.0
- **Homepage:** https://bk-hash.github.io/CCSP-Exam/
- **Purpose:** Interactive flashcard app for CCSP exam preparation.

## Architecture
- **Framework:** React (v19.1.0)
- **Build Tool:** react-scripts (v5.0.1)
- **Deployment:** GitHub Pages via `gh-pages` package
- **Testing:** @testing-library/react, jest-dom

## Key Components
- Flashcard display and navigation
- Quiz logic and scoring
- Responsive UI

## Configuration
- **Environment:** No custom environment variables required
- **Build:** `npm run build`
- **Deploy:** `npm run deploy` (publishes to GitHub Pages)
